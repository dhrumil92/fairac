# Project Name (Working Title)

**FairAC** (temporary name)

Purpose:

> To fairly divide AC electricity consumption among hostel, PG, and flat roommates based on actual participation instead of equal sharing.

---

# 1. User Roles

## Student

A resident of a room.

Can:
- Register account
- Login
- Join room
- Create room
- Participate in sessions
- View consumption
- View wallet
- View history

---

## Admin

Hostel rector / PG owner.

Can:
- Manage rooms
- Manage students
- Recharge wallets
- Monitor consumption
- Monitor sessions
- Generate reports

---

## Future Role: Super Admin

You (system owner).

Can:
- Manage hostels
- Manage admins
- Manage subscriptions

Not needed now.
